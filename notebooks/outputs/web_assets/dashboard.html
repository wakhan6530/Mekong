📊 Project Overview

Mission: Quantify hydrological impacts of upstream dams on the Mekong River and Tonlé Sap Lake using Sentinel-1 SAR satellite data.

Study Areas

Mekong Delta Vietnam

Major rice production hub, 17M people dependent

Tonlé Sap Cambodia

Southeast Asia's largest lake, critical fishery

Methodology

  • Data Source: Sentinel-1 C-band SAR (all-weather, day/night)
  • Analysis Period: 2015–2024 (10 years)
  • Baseline: Landsat 5 pre-dam era (2005–2008)
  • Key Innovation: Dual-polarization (VV + VH) to detect flooded vegetation
  • Platform: Google Earth Engine + Python scientific stack
⚠️ Critical Event: July 2019 — Jinghong Dam (China) artificial flow restriction triggered downstream ecological crisis. This dashboard tracks pre/post impacts.

🎯 Key Findings & Recommendations

Scientific Contributions

  1. Dual-polarization advantage quantified: VH detects ~20% more flooding than VV
  2. Economic impact: Hidden agricultural damage ($XXM over 10 years)
  3. Ecological threshold violations: Tonlé Sap fishery at risk (XX% of years)
  4. Dam impact proven: Decoupling from natural precipitation patterns
  5. Post-2019 intensification: Temporal coincidence with Jinghong event

Policy Recommendations

  • Implement regional early warning system using dual-pol SAR
  • Enforce ecological flow requirements for upstream dams
  • Establish Tonlé Sap minimum threshold (2,500 km²) as legal mandate
  • Create transboundary monitoring framework (China-Laos-Thailand-Cambodia-Vietnam)
  • Invest in adaptation: floating rice varieties, aquaculture diversification
🚀 Next Steps: Expand to monthly analysis, integrate additional sensors (optical + SAR fusion), develop real-time operational dashboard for Mekong River Commission.

Limitations & Uncertainties

  • SAR thresholds empirical (±2 dB → ~10% area uncertainty)
  • Economic model uses scenario-based loss factors (order-of-magnitude estimates)
  • Correlation ≠ causation (but preponderance of evidence supports dam impact)
  • 2-month composites miss intra-seasonal variability
  • Single land cover map (2020) applied to all years

🔧 Technical Details

Data Processing Pipeline

  1. Sentinel-1 GRD acquisition (Google Earth Engine)
  2. Backscatter threshold classification (VV < -16 dB, VH < -22 dB)
  3. Morphological refinement (opening + closing, 30m radius)
  4. Topographic masking (slope ≤ 5°, NASADEM)
  5. Land cover validation (ESA WorldCover 2020)
  6. Area computation (30m scale, tile processing)

Quality Assurance

  • Scene count tracking (≥5 good, 3-4 fair, <3 poor)
  • Temporal coverage assessment (distribution over analysis window)
  • Spatial consistency checks (VH ≥ VV validation)
  • Independent validation (JRC Global Surface Water: 60-80% agreement)
  • Sensitivity analysis (threshold variation: CV < 15%)
